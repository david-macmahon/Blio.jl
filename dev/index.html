Blio.jl

Breakthrough Listen I/O for Julia

Blio.jl provides I/O support for radio astronomy data file formats used by the Breakthrough Listen program. Currently it supports reading and writing:

  • SIGPROC Filterbank files (.fil) used for reduced-rate filterbank spectra.
  • GUPPI Raw files (.raw) produced by several radio telescope backends, and

Support for converting between Filterbank and HDF5 (.h5) files is provided via an optional extension that is loaded when HDF5.jl is also available.

Features

  • Read and write SIGPROC Filterbank headers and spectra
  • Read and write GUPPI Raw headers and data blocks
  • Memory map data arrays for efficient access to large files
  • Convert between GuppiRaw headers and Filterbank headers
  • Convert between .fil and .h5 formats (requires HDF5.jl)
  • Optionally load headers into DataFrames (requires DataFrames.jl)
  • Optionally wrap data arrays in DimArrays (requires DimensionalData.jl)

Installation

Blio.jl is not yet registered in the General registry. You can install it directly from the GitHub repository:

using Pkg
Pkg.add(url = "https://github.com/david-macmahon/Blio.jl.git")
